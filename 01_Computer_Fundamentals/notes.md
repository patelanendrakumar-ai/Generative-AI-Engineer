Interview Questions:-
Computer क्या है?
CPU क्या करता है?
RAM Temporary Memory क्यों कहलाती है?

Chapter 1 Summary:-
✅ Computer केवल Instructions Follow करता है।
✅ CPU सभी Instructions Execute करता है।
✅ RAM अस्थायी Memory है।
✅ SSD स्थायी Storage है।
✅ Python Code पहले Interpreter पढ़ता है, फिर CPU उसे Execute करता है।
✅ AI Systems में GPU की भूमिका बहुत महत्वपूर्ण होती है।
